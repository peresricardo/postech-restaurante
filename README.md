## Execução de testes no projeto
<hr>

- Para executar os testes unitários:

```sh
mvn test
```
- Para executar os testes integrados:
```sh
mvn test -P integration-test
```
- Para executar os testes de sistema:
```sh
mvn test -P system-test
```
